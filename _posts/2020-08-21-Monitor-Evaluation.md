# Moniter

## Cretiria

- 刷新率（Hz）亮度（nit）色深（bit）对比度 → 越高越好
- 响应时间（ms）→ 越快越好
- 点距 → 越小越清晰
- 15 寸 mbp 至少需要 85W 才能完美供电, 有 type C 接口
